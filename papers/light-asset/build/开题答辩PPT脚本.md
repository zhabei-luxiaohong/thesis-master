# 开题答辩PPT制作脚本

**设计方向**：参考 McKinsey / 顶级咨询公司风格——克制、数据说话、一页一个论断。
**配色**：深灰底(#1a1a2e) + 白色文字(#ffffff) + 红色强调(#c0392b)。
**字体**：标题用思源黑体 Bold，正文用微软雅黑。

## 第6页 · 文献坐标

标题：「本文在坐标系中的位置」（32pt，白色）

左侧：画坐标系（PowerPoint用线条工具画十字）

- X轴：左侧标「金融工具视角」，右侧标「企业战略转型视角」（红色）
- Y轴：上标「多维度评估」，下标「单一指标」

四个象限用半透明浅色填充，右上角象限稍亮。

散布8-10个灰色小圆点代表现有文献。

在右上角放一个**红色大圆点**，标「本文」。

右侧文字框：
```
定位：企业战略转型视角 × 多维度系统评估

→ 构建"交易结构-出表效果-转型效应"
  统一分析框架

→ 五维度财务穿透 + 三案例双向验证
```

## 第7页 · 三个缺口

标题：「三个研究不足」（32pt，白色）

三个横条，左侧有红色竖线（3pt）：

```
01  交易结构 → 出表效果 → 转型效应：尚未整合
    Hardin/Wu · 原野/彭 · 王宇/马 —— 分属金融学、会计学、公司财务，未形成统一框架

    Sohn→ROE · Hardin→负债率 · 吴晓波→盈利波动 —— 各从一个维度，缺少系统工具

03  正反案例双向验证缺失
    多单案例描述（万达/越秀/凯德），华润"双平台"首创实践文献空白
```

## 第8页 · 五个假设

标题：「五个研究假设」（32pt，白色）

五条，每条一个深灰横条，左侧红色序号：

```
01  H₁：REITs发行 → 经常性收入占比显著提高
02  H₂：轻资产转型 → 净负债率显著下降
03  H₃：运营能力是转型成功的必要条件
04  H₄：真实出售的效应 远大于 售后租回
05  H₅：完整资本循环 → 转型效应更持久
```

底部灰色小字：理论依据：Hardin & Wu (2010) · Sohn et al. (2013) · 原野、彭晓松 (2019) · 刘月龙 (2023)

## 第9页 · 分析框架

标题：「分析框架」（32pt，白色）

左右两栏：

**左栏** — 三层架构（三个箭头串联的灰色卡片）：
```
REITs驱动机制    →    制度功能 × 企业战略需求
        ↓
路径选择         →    华润双平台 vs 凯德资本闭环
        ↓
效应评估         →    财务效应(5维) + 运营效应(3维)
```

**右栏** — 上半：「五维度财务穿透」
```
① 资产轻量化    ② 收入结构
③ 盈利质量      ④ 现金流
⑤ 资本效率
```

下半：「五阶段路径」
```
持有 → 剥离 → 平台 → REITs → 循环
```

底部：数据来源：年报 + REITs文件 + Wind/戴德梁行/彭博

## 第10页 · 创新点与大纲

标题：「四个创新点」（32pt，白色）

四个横条，左侧红色序号：

```
① 首次系统研究"双平台模式"
   华润"商管分拆+REITs发行"国内首创，文献空白

② 构建五维度财务穿透框架
   超越ROE/负债率，资产→收入→盈利→现金流→资本效率

③ 引入反面案例双向验证
   苏宁"出表不降杠杆" ↔ 华润/凯德成功，正反对照

④ 跨国对比增强普适性
   华润(央企) ↔ 凯德(新加坡)，制度环境差异检验
```

```
绪论 → 文献综述 → 研究设计
→ 华润分析 → 凯德分析
→ 苏宁反面 → 结论建议
```
