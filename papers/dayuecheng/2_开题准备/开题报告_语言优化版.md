## 一、研究背景与意义

### 1.1 研究背景

#### 宏观背景：房地产行业深度调整与REITs市场快速发展
- **行业深度调整**：自2021年起，中国房地产行业进入深度调整阶段，多家房企面临严峻的财务困境，行业转型压力持续加大。
- **REITs政策支持体系**：2020年4月，中国证监会与国家发展改革委联合发布《关于推进基础设施领域不动产投资信托基金（REITs）试点相关工作的通知》，标志着中国公募REITs市场正式启动[1]。
- **市场规模持续扩容**：截至2024年底，中国公募REITs市场总规模已达1,661.64亿元，其中房企REITs规模为224.22亿元，占市场总额的13.5%[2]。
- **政策导向明确**：2024年REITs试点范围进一步扩展至消费基础设施领域，为商业地产资产证券化提供了新的政策路径和发展机遇。

#### 微观背景：大悦城首次年度亏损与REITs发行
- **财务困境凸显**：大悦城控股集团股份有限公司2024年首次出现年度亏损，净利润为-29.77亿元，同比下降幅度高达103.14%，财务状况显著恶化[3]。
- **现金流状况恶化**：2024年经营现金流净额为66.17亿元，同比下降37.82%；筹资现金流净流出93.14亿元，融资渠道明显受限[3]。
- **业务结构过度集中**：79.31%的营业收入依赖于商品房销售，业务结构高度单一，受房地产市场周期性波动影响显著。
- **REITs作为解决方案**：2024年5月24日，大悦城申报华夏大悦城购物中心封闭式基础设施证券投资基金，发行规模达33亿元，成为西南地区规模最大的商业地产REITs项目[4]。

### 1.2 研究意义

#### 理论意义
1. **丰富REITs理论应用场景**：本研究从受困房企的独特视角切入，拓展了REITs理论的应用边界和研究深度。
2. **深化财务困境研究维度**：系统探讨REITs作为财务困境解决方案的作用机制，丰富财务困境理论的研究内容。
3. **拓展资产证券化理论体系**：基于中国REITs市场的特殊性，探索资产证券化理论的本土化应用和创新路径。
4. **构建整合性理论框架**：建立"财务困境-REITs救援-财务改善"的整合理论框架，促进不同理论体系的交叉融合。

#### 实践意义
1. **为企业决策提供参考**：为面临财务困境的房地产企业提供REITs应用的具体决策依据和操作指引。
2. **构建投资分析框架**：为投资者提供评估房企REITs投资价值的分析工具和决策框架。
3. **支撑政策制定优化**：为监管部门完善REITs相关政策提供实践层面的参考依据和建议。
4. **指导行业转型发展**：促进房地产行业从重资产开发模式向轻资产运营模式的战略转型。

### 1.3 研究创新点

#### 视角创新
- **首次聚焦受困房企视角**：打破现有研究多关注健康企业的局限，从受困房企的独特视角探究REITs的救援效应。
- **填补关键研究空白**：现有文献中缺乏对受困房企REITs应用的系统研究，本研究致力于填补这一重要学术空白。

#### 方法创新
- **案例研究驱动的方法论**：采用Yin(2013)案例研究框架作为核心方法论，聚焦"REITs如何缓解财务困境"的"How/Why"问题，而非早期DID为主的研究设计。
- **分析方法融合创新**：构建"四机制为核心+5D为补充"的整合分析框架（融资结构优化/现金流改善/资产负债重构/运营效率提升为核心，筛选/结构/运营/风险/跨境为补充视角）。
- **混合证据链构建**：建立定量财务数据趋势对比+定性文本证据+行业参照案例的三维证据链，提升研究结论的稳健性和可信度。
- **方法论优化说明**：
  - **DID降级为探索性分析**：早期工作流中DID作为核心方法的定位已调整，3家企业×6年=18个观测值不满足DID大样本要求，DID仅作为补充分析并标注为"探索性"。
  - **案例设计修正**：万科/保利从"控制组"改为"对比参照案例"，提供不同困境程度的参照分析，而非DID的控制组设计。
  - **分析框架统一**：避免了早期工作流中5D框架与四机制的分析维度冲突，实现逻辑一致性。

#### 应用创新
- **提供直接决策支持**：基于真实案例分析，为受困房企提供具有可操作性的REITs应用建议和决策支持。
- **拓展行业应用价值**：研究成果可直接应用于房企转型实践，推动行业创新发展。
- **增强政策应用价值**：为REITs政策的完善和优化提供实证依据，提升政策制定的科学性和针对性。

## 二、文献综述

### 2.1 REITs理论研究

#### 2.1.1 REITs基本理论
- **定义与核心特征**：不动产投资信托基金（REITs）是一种通过发行收益凭证汇集投资者资金，由专业投资管理机构进行不动产投资经营管理，并将投资收益按比例分配给投资者的基金组织形式[5]。
- **全球发展历程**：REITs于1960年在美国正式诞生，2000年后在全球范围内快速发展，2020年在中国正式启动试点工作，标志着中国REITs市场的正式建立[6]。
- **基本运作模式**：遵循"资产收购→专业化运营→租金收益→投资者分红"的标准化运作流程，形成完整的商业闭环。

#### 2.1.2 REITs财务效应研究
- **融资效应显著**：Chan等人（2013）的研究证实，REITs发行能够显著降低企业的融资成本，改善融资结构[7]。
- **资产负债结构优化**：Liu和Sun（2021）的研究结果表明，REITs发行有助于优化企业资本结构，降低财务风险[8]。
- **企业价值提升效应**：众多研究表明，REITs发行通常带来显著的股东财富效应，提升企业市场价值[9]。

### 2.2 财务困境理论研究

#### 2.2.1 财务困境概念与成因
- **概念界定**：Altman（1968）率先提出财务困境概念，将其界定为企业无法偿还到期债务的财务状态[10]。
- **成因多维分析**：主要包括过度扩张、经营效率低下、宏观环境变化等多方面因素的综合作用[11]。
- **主要解决路径**：通常包括资产重组、债务重组、业务转型等多种方式[12]。

#### 2.2.2 中国房企财务困境特征
- **行业特殊性突出**：高杠杆运作、投资周期长、政策敏感性强是房地产行业的显著特征。
- **典型困境表现**：流动性危机频发、债务违约风险增加、资产减值压力加大。
- **战略转型需求**：行业亟需从传统的重资产开发模式向轻资产运营模式转型。

### 2.3 REITs与财务困境相关研究

#### 2.3.1 现有研究评述
- **研究覆盖不足**：现有研究大多关注REITs对财务状况良好企业的影响，缺乏对受困企业的专门研究[13]。
- **机制研究薄弱**：REITs影响财务困境的具体作用机制缺乏深入系统的研究[14]。
- **本土化研究有限**：基于中国REITs市场特殊性的案例研究相对匮乏[15]。

#### 2.3.2 研究缺口识别
1. **视角缺口**：缺乏从受困房企视角的REITs应用研究。
2. **机制缺口**：REITs缓解财务困境的具体机制不够明确。
3. **实证缺口**：基于中国本土案例的实证研究相对不足。
4. **比较缺口**：不同类型房企REITs应用的比较研究较为缺乏。

### 2.4 文献评述与研究定位

#### 2.4.1 理论定位
- **研究领域交叉**：本研究定位于房地产金融、企业财务困境与资产证券化三个研究领域的交叉地带。
- **理论贡献明确**：旨在整合REITs理论与财务困境理论，填补现有研究的空白领域。

#### 2.4.2 实践定位
- **研究对象聚焦**：以大悦城控股为典型案例，深入探究REITs对受困房企的变革效应。
- **研究价值显著**：研究成果有望为企业决策、投资分析、政策制定提供重要的实证依据。

## 三、研究内容与方法

### 3.1 研究内容

#### 3.1.1 核心研究问题
**REITs发行对受困房企的财务困境缓解效应如何？**

具体分解为四个子问题：
1. REITs发行如何影响受困房企的资产负债结构？
2. REITs发行如何改善受困房企的现金流状况？
3. REITs发行对受困房企的市场价值有何影响？
4. REITs影响财务困境的具体作用机制是什么？

#### 3.1.2 研究目标
1. **理论目标**：构建REITs影响财务困境的理论分析框架。
2. **实证目标**：验证大悦城REITs发行的财务效应和市场反应。
3. **比较目标**：分析不同类型房企REITs应用的效果差异。
4. **应用目标**：提出受困房企REITs应用的实践建议。

### 3.2 研究方法

#### 3.2.1 核心方法论：Yin案例研究框架
本研究采用Robert K. Yin（2013）的案例研究法作为核心方法论，基于以下设计：

| Yin框架要素 | 本研究应用 | 理论依据 |
|------------|------------|----------|
| **研究问题类型** | "How/Why"型问题：REITs如何缓解财务困境 | 案例研究最适合回答"How/Why"的机制性问题 |
| **理论主张** | 四机制理论框架（融资/现金流/资产负债/运营） | 提供理论预期，指导证据收集 |
| **分析单位** | 大悦城（单案例深度）+ 万科/保利（简要参照） | 大悦城为典型代表案例 |
| **逻辑连接** | 模式匹配（理论预期 vs. 实证发现） | 检验理论框架的解释力 |

#### 3.2.2 方法组合设计
**主方法 + 辅助方法矩阵**：
| 方法类型 | 具体方法 | 定位 | 数据需求 | 产出 |
|----------|----------|------|----------|------|
| **核心方法** | Yin案例研究法 | 理论框架检验 | 大悦城深度数据 | 机制分析 |
| **辅助方法1** | 财务指标趋势对比 | 财务效应检验 | 时间序列财务数据 | 趋势图表 |
| **辅助方法2** | 事件研究法 | 市场反应分析 | 股价事件数据 | CAR值 |
| **辅助方法3** | 比较分析法 | 参照分析 | 万科/保利数据 | 对比视角 |
| **探索性方法** | DID模型 | 补充探索分析 | 面板数据（限） | 探索性结果 |

#### 3.2.3 方法论优化说明
- **DID定位调整**：原设计中DID作为核心方法，现调整为**探索性补充分析**，需明确标注样本量限制（3家企业×6年=18个观测值不满足大样本要求）。
- **案例设计修正**：万科/保利从"控制组"改为"**对比参照案例**"，提供不同困境程度的参照分析，避免规模差异带来的可比性问题。
- **分析框架统一**：采用"**四机制为核心+5D为补充**"的统一框架，避免分析维度冲突。

### 3.3 研究框架

#### 3.3.1 理论框架：四机制分析框架
本研究的核心理论框架是"REITs对财务困境的缓解效应四机制"：

```
财务困境 → REITs发行 → 四机制效应 → 财务改善
    ↓
融资结构优化机制：
    1. 降低融资成本
    2. 优化债务期限结构
    3. 增加融资渠道多样性
    
现金流改善机制：
    1. 提升租金收入稳定性
    2. 改善经营性现金流
    3. 降低财务费用支出
    
资产负债重构机制：
    1. 提升资产周转效率
    2. 优化资本结构
    3. 改善偿债能力指标
    
运营效率提升机制：
    1. 专业化运营管理
    2. 降低经营成本
    3. 提升投资回报率
```

**框架定位**：
- **核心框架**：四机制分析（融资/现金流/资产负债/运营）——回答研究核心问题
- **补充视角**：5D框架（筛选/结构/运营/风险/跨境）——提供REITs产品设计维度的补充分析
- **分析方法**：模式匹配（理论预期 vs. 实证发现）——检验理论框架的解释力

#### 3.3.2 实证框架：时间序列+案例对比
```
研究问题 → 理论框架构建 → 案例选择设计 → 数据收集验证
    ↓
实证分析路径：
1. **趋势对比分析**（时间维度）
   - 财务指标时间序列分析：2019-2024年
   - REITs发行前后对比：2023 vs 2024 vs 2025
   
2. **机制检验分析**（框架维度）
   - 四机制框架检验：融资/现金流/资产负债/运营
   - 模式匹配度评估：理论预期 vs. 实证发现
   
3. **参照对比分析**（案例维度）
   - 大悦城深度分析：纵向时间序列
   - 万科/保利简要参照：横向对比视角
   
4. **探索性补充分析**（方法维度）
   - DID模型探索：标注样本量限制
   - 稳健性检验：多种模型敏感性分析
    ↓
研究结论 → 理论贡献 → 实践启示 → 政策建议
```

### 3.4 技术路线

#### 3.4.1 研究实施路线
```
文献调研 → 理论框架构建 → 研究设计 → 数据收集与验证 → 实证分析 → 结果解释 → 研究结论
```


## 四、研究设计

### 4.0 方法论定位说明
本研究采用**Yin案例研究法**作为核心方法论，聚焦"REITs如何缓解财务困境"的"How/Why"问题。基于方法论交叉审查优化，调整如下：

1. **方法主次调整**：
   - ✅ **核心方法**：Yin案例研究法（理论验证型）+ 四机制分析框架
   - ✅ **辅助方法**：财务指标趋势对比 + 事件研究法 + 比较分析法
   - ⚠️ **探索性方法**：DID模型（仅作为补充分析，标注样本量限制）

2. **样本设计修正**：
   - ✅ **核心案例**：大悦城（单案例深度分析）
   - ✅ **参照案例**：万科/保利（简要对比参照，非DID控制组）
   - ❌ **控制组设计**：放弃（规模差异过大，万科1.9万亿 vs 大悦城0.2万亿）

3. **分析框架统一**：
   - ✅ **核心框架**：四机制（融资/现金流/资产负债/运营）为核心
   - ✅ **补充视角**：5D框架（筛选/结构/运营/风险/跨境）为补充
   - ❌ **框架冲突**：避免早期工作流中5D与四机制的分析维度冲突

### 4.1 样本选择

#### 4.1.1 处理组：大悦城控股集团股份有限公司
- **选择依据**：2024年首次出现年度亏损，属于典型的财务困境案例。
- **核心事件**：2024年5月24日申报华夏大悦城购物中心封闭式基础设施证券投资基金。
- **数据需求**：2019-2024年完整的年度财务数据。

#### 4.1.2 对比参照案例设计（非DID控制组）
基于方法论优化，万科/保利定位调整为**对比参照案例**，而非DID控制组：

| 企业 | 案例定位 | 角色说明 | 分析方法 |
|------|----------|----------|----------|
| **大悦城** | **核心案例** | 深度分析对象，验证REITs对财务困境的缓解机制 | Yin案例研究法 + 四机制框架检验 |
| **万科** | **转型参照** | 提供大规模房企REITs转型实践的参照视角 | 简要比较分析，非因果推断 |
| **保利** | **创新参照** | 提供早期REITs创新尝试的参照视角 | 简要比较分析，非因果推断 |

**选择逻辑与定位说明**：
1. **规模差异问题**：万科总资产1.9万亿 vs 大悦城0.2万亿（10倍差异），不符合DID控制组的可比性要求
2. **处理效应混淆**：万科2024年已发行中金印力REIT，自身已受"处理"，不能作为"未处理"控制组
3. **研究设计调整**：从因果推断（DID）转为**模式匹配与参照分析**，更符合案例研究逻辑
4. **分析深度差异**：大悦城深度分析（时间序列+机制检验） vs 万科/保利简要参照（提供对比视角）

### 4.4 核心分析方法

#### 4.4.1 四机制框架检验模型
**核心分析路径**：基于Yin案例研究法的模式匹配检验

```
1. 理论预期构建（基于四机制框架）：
   融资结构优化 → 融资成本降低、债务期限优化
   现金流改善 → 经营现金流改善、财务费用降低
   资产负债重构 → 资产周转提升、资本结构优化
   运营效率提升 → 专业运营管理、投资回报提升

2. 实证发现收集（基于时间序列数据）：
   - 财务指标时间序列：2019-2024年
   - REITs发行前后对比：2023 vs 2024 vs 2025
   - 关键事件分析：REITs申报、发行、运营

3. 模式匹配度评估：
   - 定性评估：理论预期与实证发现的一致性程度
   - 定量评估：趋势方向一致性、变化幅度合理性
   - 综合判断：四机制框架的解释力和适用性
```

**具体检验方法**：
1. **趋势对比分析**（时间维度）：
   ```
   ΔY_t = Y_{t,post} - Y_{t,pre}   # REITs发行前后财务指标变化
   ΔY_annual = Y_{t+1} - Y_t       # 年度变化趋势
   ```

2. **参照对比分析**（案例维度）：
   ```
   ΔY_diff = Y_{treatment} - Y_{reference}   # 大悦城 vs 万科/保利差异
   ```

3. **机制分解检验**（框架维度）：
   - 融资结构：融资成本率、债务期限结构、融资渠道数
   - 现金流：经营现金流比率、财务费用率、现金持有水平
   - 资产负债：资产周转率、资产负债率、偿债能力指标
   - 运营效率：管理费用率、投资回报率、专业化水平

#### 4.4.2 探索性补充分析方法
**DID模型探索性应用**：
- **定位说明**：非核心因果推断方法，仅作为探索性补充分析
- **样本限制**：3家企业×6年=18个观测值，远低于DID大样本要求
- **模型表述**：如进行回归分析，需明确标注"探索性"并说明统计推断的局限性
- **结果解释**：谨慎解释系数，避免过度强调因果效应

**敏感性分析策略**：
1. **变量替换**：使用不同财务指标衡量相同概念
2. **期间调整**：调整REITs事件窗口长度
3. **模型设定**：尝试不同函数形式和估计方法
4. **异常值处理**：排除极端值检验结果稳健性

#### 4.4.2 事件研究法模型
```
CAR_i = α + βREIT_EVENT_i + γX_i + ε_i
```
 sowie
- `CAR_i`：事件窗口累计异常收益率
- `REIT_EVENT_i`：REITs发行事件哑变量
- `X_i`：企业特征控制变量

#### 4.4.3 机制检验模型
```
M_it = α + β₁REIT_DUMMY_it + γX_it + ε_it
Y_it = α + β₁REIT_DUMMY_it + β₂M_it + γX_it + ε_it
```
 sowie
- `M_it`：中介变量（融资成本、现金流改善等）
- 采用Bootstrap法检验中介效应显著性

### 4.5 预期实证结果

#### 4.5.1 核心结果预期
| 假设 | 预期效应 | 显著性 | 经济意义 |
|------|----------|--------|----------|

## 七、研究局限与改进方向

### 7.1 研究局限性

#### 7.1.1 样本规模限制
- **样本数量有限**：仅研究三家企业，统计效力相对有限。
- **行业代表性**：虽然选择了不同类型企业，但行业覆盖范围有限。
- **时间跨度限制**：REITs发行时间较短，长期效应观察不足。

#### 7.1.2 方法局限性
- **内生性问题**：可能存在反向因果关系等内生性问题。
- **遗漏变量问题**：可能遗漏重要的控制变量。
- **数据局限性**：部分财务数据可能不完整或不准确。

#### 7.1.3 外部效度限制
- **案例特殊性**：大悦城案例可能具有一定的特殊性。
- **行业特殊性**：房地产行业的特殊性可能影响研究的普适性。
- **政策环境特殊性**：中国REITs政策环境的特殊性需要考虑。

### 7.2 改进方向

#### 7.2.1 研究设计改进
1. **扩大样本规模**：增加更多房企案例，提高研究的代表性和普遍性。
2. **延长研究期间**：持续跟踪REITs发行后的长期效应和动态变化。
3. **深化机制分析**：采用更精细的研究方法检验作用机制。

#### 7.2.2 方法改进
1. **采用更高级模型**：使用动态面板模型、倾向得分匹配等高级方法。
2. **多源数据验证**：增加访谈、问卷调查等定性数据验证。
3. **稳健性检验强化**：进行更全面系统的稳健性检验。

#### 7.2.3 应用拓展
1. **行业拓展研究**：研究REITs在其他受困行业中的应用效果。
2. **国际比较研究**：对比分析不同国家REITs市场的特点和应用效果。
3. **政策影响研究**：研究政策变化对REITs效果的影响机制。
