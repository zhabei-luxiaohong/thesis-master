# DID模型基线数据分析计划

## 一、研究设计框架

### 1.1 核心研究问题
- **研究问题**：REITs发行对受困房企财务困境的缓解效应
- **研究假设**：
  - H1：REITs发行能显著降低受困房企资产负债率
  - H2：REITs发行能显著改善受困房企经营现金流
  - H3：REITs发行能显著提升受困房企市场价值

### 1.2 双重差分模型（DID）设计
```
处理组：大悦城控股（000031.SZ）
控制组：万科企业（000002.SZ）、保利发展（600048.SH）
事件时点：2024年5月24日（华夏大悦城REIT申报）
研究期间：2019年1月1日-2024年12月31日
```

## 二、变量定义与数据要求

### 2.1 主要变量定义

#### 被解释变量（Y）：
| 变量 | 定义 | 数据来源 | 当前状态 |
|------|------|----------|----------|
| **资产负债率（LEV）** | 总负债/总资产 | 公司年报 | ✅ 可用 |
| **经营现金流比率（CFO_R）** | 经营现金流/营业收入 | 公司年报 | ✅ 可用 |
| **净利润率（PROFIT）** | 净利润/营业收入 | 公司年报 | ✅ 可用 |

#### 解释变量：
| 变量 | 定义 | 数据来源 | 当前状态 |
|------|------|----------|----------|
| **Treat** | 处理组=1（大悦城），控制组=0（万科、保利） | 公司列表 | ✅ 可用 |
| **Post** | 事件后=1（2024.5.24之后），事件前=0 | 日期 | ✅ 可用 |
| **Treat×Post** | DID交互项（核心解释变量） | 计算 | ✅ 可用 |

#### 控制变量：
| 变量 | 定义 | 数据来源 | 当前状态 |
|------|------|----------|----------|
| **公司规模（SIZE）** | Ln(总资产) | 公司年报 | ⚠️ 部分可用 |
| **成长性（GROWTH）** | 营业收入增长率 | 公司年报 | ⚠️ 部分可用 |
| **盈利能力（ROA）** | 净利润/总资产 | 公司年报 | ⚠️ 部分可用 |
| **行业因素（INDUSTRY）** | 房地产行业虚拟变量 | 行业分类 | ✅ 可用 |
| **年度效应（YEAR）** | 年度虚拟变量 | 年份 | ✅ 可用 |

## 三、基线数据分析步骤

### 3.1 数据准备阶段（第1周）

#### 任务1：数据收集与验证
```
1.1 下载大悦城2019-2024年官方年报PDF（巨潮资讯网）
1.2 提取完整财务数据：
    - 总资产、总负债（计算资产负债率）
    - 经营现金流（计算现金流比率）
    - 净利润（计算净利润率）
    - 营业收入（计算成长性）
1.3 下载万科2019-2024年完整年报
1.4 下载保利2019-2024年完整年报
1.5 建立统一数据格式（Excel/CSV）
```

#### 任务2：数据处理与清洗
```
2.1 检查数据异常值（±3标准差）
2.2 处理缺失数据（插值/前向填充）
2.3 计算关键财务比率
2.4 构建面板数据结构（公司×年份）
2.5 变量标准化处理
```

### 3.2 描述性统计分析（第2周）

#### 任务3：总体描述性统计
```
3.1 计算全部变量的均值、标准差、最小值、最大值
3.2 绘制处理组与控制组的时间趋势图
3.3 检验变量分布的正态性
```

#### 任务4：分组比较分析
```
4.1 事件前后分组比较（t检验）
4.2 处理组与控制组对比分析
4.3 关键变量的年度变化趋势
```

### 3.3 DID模型构建（第3周）

#### 任务5：基准DID模型估计
```
5.1 构建标准DID模型：
    Y_it = α + β1Treat_i + β2Post_t + β3(Treat_i×Post_t) + ε_it
    
5.2 估计三种被解释变量模型：
    - 模型1：资产负债率（LEV）
    - 模型2：经营现金流比率（CFO_R）
    - 模型3：净利润率（PROFIT）
```

#### 任务6：控制变量纳入
```
6.1 加入公司层面控制变量
6.2 加入行业和时间固定效应
6.3 检验模型稳健性
```

### 3.4 平行趋势检验（第4周）

#### 任务7：平行趋势检验
```
7.1 绘制事件前趋势图
7.2 进行动态DID估计
7.3 检验处理组与控制组在事件前的平行趋势
```

#### 任务8：稳健性检验
```
8.1 安慰剂检验
8.2 替换控制组检验
8.3 调整研究期间检验
```

## 七、研究方法论要点

### 7.1 计量模型选择依据

#### 选择DID模型的原因：
1. **因果识别优势**：有效分离REITs的净效应
2. **数据适用性**：处理组+控制组结构清晰
3. **方法成熟度**：经济学领域广泛应用
4. **结果可信度**：通过平行趋势检验增强可信度

### 7.2 模型设定注意事项

#### 关键设定考虑：
1. **时间效应**：必须控制年度时间趋势
2. **公司固定效应**：控制不随时间变化的公司特征
3. **异方差调整**：使用稳健标准误
4. **聚类标准误**：在公司层面聚类

### 7.3 内生性问题处理

#### 可能的内生性来源：
1. **样本选择偏误**：REITs发行可能非随机
2. **遗漏变量偏误**：未观测到的影响因素
3. **双向因果**：财务困境可能导致REITs发行

#### 处理方法：
1. **平行趋势检验**：检验事件前的可比性
2. **安慰剂检验**：排除随机性影响
3. **控制变量加入**：尽可能控制相关因素

## 八、数据质量控制

### 8.1 数据验证流程

1. **来源验证**：所有数据至少两个独立来源验证
2. **逻辑检验**：财务比率在合理范围内
3. **一致性检验**：不同年份数据衔接合理
4. **跨平台验证**：巨潮资讯网+Wind数据库交叉验证

### 8.2 数据处理标准

1. **缺失值处理**：使用前向填充或行业均值插补
2. **异常值处理**：±3标准差之外的数据进行缩尾处理
3. **变量转换**：使用自然对数转换处理规模变量
4. **标准化处理**：控制变量进行标准化处理
