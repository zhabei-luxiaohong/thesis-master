# 证据三角形验证实施计划（基于v2.1工作流）

## 1. 验证框架设计

### 1.1 证据三角形概念
```
            证据三角形
           /          \
    定量数据          定性数据
   (财务指标)       (文本材料)
      |                |
  年报/季报        公告/研报
  REITs说明书      管理层讨论
  行业统计数据      政策文件
```

### 1.2 验证目标
- **数据可信度**：确保每个关键数据点有至少两个独立来源验证
- **方法严谨性**：建立系统化的数据验证流程
- **研究可复现**：提供完整的数据溯源链
- **结论可靠性**：增强研究结论的说服力和可信度

### 1.3 验证原则
1. **官方优先**：官方财报数据优先级最高
2. **多源验证**：关键数据点需要两个以上来源验证
3. **透明记录**：完整记录数据来源和验证过程
4. **不一致处理**：明确不一致数据的处理原则

## 4. 验证流程设计

### 4.1 标准化验证流程

```
步骤1：数据收集 → 步骤2：初步分类 → 步骤3：来源验证 → 步骤4：一致性检查 → 步骤5：最终确定
```

#### 步骤1：数据收集
- **多渠道收集**：从至少两个渠道收集同一数据点
- **完整记录**：记录数据来源、获取时间、原始格式
- **初步整理**：整理成统一格式的表格

#### 步骤2：初步分类
- **来源分类**：区分官方、媒体、数据库等来源类型
- **质量评估**：初步评估数据质量和可信度
- **优先级排序**：确定验证的优先级

#### 步骤3：来源验证
- **官方验证**：核对官方来源的数据
- **交叉验证**：不同来源数据的对比
- **权威性确认**：确认来源的权威性和可信度

#### 步骤4：一致性检查
- **数据对比**：对比不同来源的同一数据点
- **差异分析**：分析数据差异的原因
- **一致性判断**：判断数据是否一致

#### 步骤5：最终确定
- **数据选择**：选择最可信的数据版本
- **处理说明**：记录数据选择和处理的理由
- **引用准备**：准备规范的引用格式

### 4.2 不一致数据处理流程

```
发现不一致 → 分析差异原因 → 确定处理方式 → 记录处理说明 → 最终数据确定
```

#### 不一致类型及处理：
1. **财报vs媒体报道不一致**
   - **处理**：优先采用财报数据
   - **说明**：注明媒体报道作为参考，分析差异原因

2. **不同年份统计口径变化**
   - **处理**：进行可比性调整
   - **说明**：注明调整事项和调整方法

3. **临时会计政策变更**
   - **处理**：说明影响程度
   - **说明**：必要时进行数据调整

4. **市场传闻vs官方数据**
   - **处理**：以官方数据为准
   - **说明**：注明传闻作为市场反应参考

## 7. 质量控制

### 7.1 质量评估指标

| 指标 | 评估标准 | 目标值 | 检查方法 |
|------|----------|--------|----------|
| **数据覆盖率** | 关键数据点验证比例 | ≥95% | 检查验证状态表 |
| **来源可靠性** | 官方来源使用比例 | ≥80% | 统计来源类型 |
| **一致性率** | 数据一致性比例 | ≥90% | 分析不一致数据 |
| **记录完整性** | 验证记录完整比例 | 100% | 检查验证记录 |
| **时效性** | 数据更新及时性 | 及时更新 | 检查更新时间 |

### 7.2 质量控制措施

1. **双人验证**：重要数据点由两人独立验证
2. **定期检查**：每周进行数据质量检查
3. **交叉核对**：不同来源数据的交叉核对
4. **导师审核**：关键数据由导师审核确认
5. **自动化检查**：使用脚本自动化检查数据一致性

### 7.3 质量检查清单

| 检查项目 | 检查内容 | 检查标准 | 检查频率 |
|----------|----------|----------|----------|
| 数据完整性 | 所有数据点是否有验证记录 | 100%有记录 | 每周 |
| 来源权威性 | 是否优先使用官方来源 | 官方来源≥80% | 每周 |
| 引用规范性 | 引用格式是否规范统一 | 符合规范要求 | 每章 |
| 一致性检查 | 不同来源数据是否一致 | 一致性≥90% | 每周 |
| 文档完整性 | 验证文档是否齐全 | 所有文档齐全 | 每月 |

### 8.1 主要风险

| 风险类型 | 风险描述 | 可能性 | 影响程度 |

| 风险 | 应对策略 | 具体措施 |
