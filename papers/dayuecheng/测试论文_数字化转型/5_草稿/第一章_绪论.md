# 第一章 绪论

## 1.1 研究背景与问题提出

### 1.1.1 行业背景：零售业的数字化浪潮

中国零售行业正经历一场深刻变革。2024年，社会消费品零售总额达45万亿元，其中线上零售占比攀升至28.5%，较2019年提升8.3个百分点。这一数字背后，是传统零售企业面临的严峻现实：电商分流客源、租金人力成本攀升、消费者需求日益多元化。

传统零售企业站在转型的十字路口。一方面，数字化转型已成行业共识，沃尔玛、家乐福等国际巨头纷纷加码数字化布局；另一方面，转型投入巨大，效果却参差不齐。部分企业投入数亿元却收效甚微，部分企业则通过数字化实现逆势增长。

这种分化现象引发了一个核心问题：数字化转型如何影响传统零售企业的绩效？投入与回报之间存在怎样的传导机制？

### 1.1.2 企业背景：永辉超市的转型之路

永辉超市成立于2001年，以生鲜经营见长，是中国生鲜超市的龙头企业。截至2024年底，永辉在全国拥有超过1000家门店，年营收约900亿元。

2017年，永辉启动数字化转型，战略定位为"到店+到家"双线融合。此后六年间，永辉在数字化领域累计投入超50亿元，涵盖IT基础设施建设、数据中台搭建、供应链数字化、会员体系升级等多个方面。

然而，转型之路并非坦途。2020-2022年，永辉连续三年亏损，累计亏损额超过80亿元。市场质疑声四起：数字化投入是否值得？转型方向是否正确？

2023年，永辉实现扭亏为盈，归母净利润达2.8亿元。2024年上半年，业绩进一步改善，毛利率同比提升2.3个百分点。这一转折是否意味着数字化效应开始显现？

永辉的转型历程，为研究数字化转型与绩效关系提供了一个完整的案例样本。

### 1.1.3 问题提出

基于上述背景，本研究聚焦以下核心问题：

**核心问题**：数字化转型如何影响传统零售企业的绩效？

具体而言，本研究试图回答以下子问题：

1. 永辉超市数字化转型的投入结构与路径是什么？
2. 数字化转型对永辉超市财务绩效的影响如何？是否存在滞后效应？
3. 数字化→绩效的传导机制是什么？运营效率是否为关键中介变量？
4. 转型过程中的关键成功因素有哪些？

## 1.2 研究意义

### 1.2.1 理论意义

本研究在理论层面具有以下贡献：

**第一，验证资源基础观在数字化情境下的适用性。** 传统资源基础观强调有形资源（土地、设备、资金）的战略价值。在数字化时代，数据、算法、平台等无形资源是否成为新的竞争优势来源？本研究通过永辉案例，探讨数字化资源的战略属性。

**第二，丰富动态能力理论的实证研究。** 动态能力理论认为，企业通过感知、捕获、重构三种能力适应环境变化。永辉的数字化转型，本质上是动态能力的构建过程。本研究将验证数字能力的构成维度及其对绩效的影响路径。

**第三，构建数字化转型→绩效的传导机制模型。** 现有研究多关注数字化与绩效的直接关系，对传导机制的探讨相对不足。本研究引入运营效率作为中介变量，揭示数字化如何通过改善运营效率来提升财务绩效。

### 1.2.2 实践意义

本研究在实践层面具有以下价值：

**第一，为传统零售企业数字化转型提供决策参考。** 永辉的转型历程，既有成功经验，也有失败教训。本研究系统梳理永辉的转型路径，提炼可复制的成功要素，为其他企业提供借鉴。

**第二，揭示转型过程中的关键成功因素。** 数字化转型是一项系统工程，涉及技术、组织、战略多个层面。本研究通过案例分析，识别影响转型成效的关键因素，帮助企业规避常见陷阱。

**第三，提出阶段性转型路径建议。** 数字化转型非一日之功，需要分阶段推进。本研究基于永辉的转型周期，划分不同阶段，提出各阶段的重点任务和预期目标。

### 1.3.2 研究框架

本研究构建"投入-传导-绩效"的分析框架：

```
数字化转型投入
    │
    ├── 技术投入：IT基础设施、数据平台、智能设备
    ├── 组织投入：数字化人才、流程重组、组织变革
    └── 战略投入：商业模式创新、生态合作
           │
           ▼
传导机制
    │
    ├── 运营效率：库存周转率、损耗率、人效
    ├── 客户价值：体验优化、复购率、客单价
    └── 供应链：采购效率、配送时效、协同能力
           │
           ▼
绩效表现
    │
    ├── 财务绩效：营收增长、利润率、ROE、现金流
    └── 非财务绩效：市场份额、客户满意度、品牌价值
```

## 1.4 研究方法与技术路线

### 1.4.1 研究方法

本研究采用案例研究法，具体遵循Yin（2018）案例研究框架：

**案例类型**：单案例研究（嵌入性分析）。选择永辉超市作为研究对象，理由有三：一是永辉数字化转型周期完整（2017-2024），数据覆盖充分；二是永辉为行业龙头，具有典型性和代表性；三是永辉公开信息丰富，数据可得性高。

**分析单元**：企业层面（永辉整体）、业务层面（生鲜、日用品、到家业务）、时间层面（2017-2024年各阶段）。

**数据来源**：文档证据（年报、公告、研究报告）、财务数据（Wind数据库）、运营数据（年报披露）、公开报道（媒体报道、高管访谈）。

**分析方法**：财务分析法（盈利能力、运营能力、偿债能力、成长能力）、事件研究法（关键事件的市场反应）、时序分析法（转型前后绩效对比）。

### 1.4.2 技术路线

本研究技术路线如下：

第一阶段：文献综述与理论框架构建。系统梳理数字化转型、零售行业数字化、绩效影响机制相关文献，构建"投入-传导-绩效"分析框架。

第二阶段：研究设计与数据收集。确定案例研究对象，设计数据收集方案，建立案例研究数据库。

第三阶段：案例分析。对永辉超市进行深度案例分析，描述转型投入与路径，分析绩效影响，验证传导机制。

第四阶段：对比分析与讨论。选取对比案例，揭示数字化转型的共性与差异，深化研究发现。

第五阶段：结论与建议。总结研究结论，提出理论与实践启示，撰写研究报告。

## 1.5 研究创新点

本研究在以下方面有所创新：

**理论层面**：构建数字化转型→绩效的传导机制模型，引入运营效率作为关键中介变量，揭示数字化如何通过改善运营效率来提升财务绩效。
