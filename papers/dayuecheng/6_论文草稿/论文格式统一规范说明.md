# 📐 MBA论文格式统一规范说明

**生效时间**：2026年4月15日  
**适用范围**：论文所有章节、附录、参考文献  
**执行标准**：MBA学术论文标准、GB/T 7714-2015

---

## 🎯 **一、总体格式要求**

### **1.1 文档结构**
```
论文标题
作者信息
摘要
关键词
目录
第一章 绪论
第二章 文献综述
第三章 研究设计
第四章 大悦城案例分析
第五章 对比案例分析
第六章 结论与建议
参考文献
致谢
附录
```

### **1.2 页面设置**
- **纸张大小**：A4
- **页边距**：上3.0cm，下2.5cm，左2.5cm，右2.5cm
- **页眉**：左侧"REITs对受困房企的变革效应研究"，右侧页码
- **页脚**：页码居中（从第一章开始）

### **1.3 字体字号**
| 内容 | 字体 | 字号 | 说明 |
|------|------|------|------|
| **论文标题** | 黑体 | 小二号 | 居中，加粗 |
| **一级标题** | 黑体 | 四号 | 居中，加粗 |
| **二级标题** | 黑体 | 小四号 | 左对齐，加粗 |
| **三级标题** | 黑体 | 五号 | 左对齐，加粗 |
| **正文** | 宋体 | 小四号 | 两端对齐 |
| **表格/图表标题** | 黑体 | 五号 | 居中，加粗 |
| **表格内容** | 宋体 | 五号 | 居中对齐 |
| **脚注** | 宋体 | 小五号 | 单倍行距 |
| **参考文献** | 宋体 | 五号 | 悬挂缩进 |

---

## 📝 **二、标题格式规范**

### **2.1 标题层级**
```
# 第一章 绪论
## 1.1 研究背景
### 1.1.1 宏观环境分析
```

### **2.2 编号规则**
- **一级标题**：第X章 XXX
- **二级标题**：X.X XXX（如：1.1、2.3）
- **三级标题**：X.X.X XXX（如：1.1.1、2.3.2）

### **2.3 格式要求**
1. **空格要求**：编号后空一格再写标题内容
2. **标点要求**：标题末尾不加标点符号
3. **对齐要求**：一级标题居中，二级标题左对齐
4. **换行要求**：标题后空一行再写正文

---

## 📊 **三、表格格式规范**

### **3.1 表格编号**
- **编号格式**：表X-X 或 表X.X
- **统一要求**：全文档统一使用一种格式
- **建议格式**：表4.1（第四章第一个表）

### **3.2 表格标题**
- **位置**：表格上方，居中
- **格式**：加粗，五号黑体
- **内容**：简洁明确，说明表格内容

### **3.3 表格内容**
| 要求 | 具体内容 |
|------|----------|
| **边框** | 三线表（上下粗线，中间细线） |
| **对齐** | 文字左对齐，数字居中对齐 |
| **单位** | 在表头注明单位 |
| **数据来源** | 表格下方注明"数据来源：" |
| **脚注** | 使用上标数字标注说明 |

### **3.4 示例**
```
表4.1 大悦城控股基本情况（截至2025年）

| 项目 | 内容 |
|------|------|
| 公司名称 | 大悦城控股集团股份有限公司 |
| 股票代码 | 000031.SZ（深交所） |

数据来源：大悦城控股2025年半年度报告，巨潮资讯网
```

## 📈 **四、图表格式规范**

### **4.1 图表编号**
- **编号格式**：图X-X 或 图X.X
- **统一要求**：全文档统一使用一种格式
- **建议格式**：图3.1（第三章第一个图）

### **4.2 图表标题**
- **位置**：图表下方，居中
- **格式**：加粗，五号黑体
- **内容**：简洁明确，说明图表内容

### **4.3 图表要求**
| 要求 | 具体内容 |
|------|----------|
| **清晰度** | 分辨率300dpi以上 |
| **颜色** | 学术论文建议使用黑白灰 |
| **字体** | 图表内字体不小于8pt |
| **坐标轴** | 必须有标签和单位 |
| **图例** | 位置合理，清晰可见 |

### **4.4 数据来源标注**
```
图4.1 大悦城净利润变化趋势（2019-2025年）
数据来源：大悦城控股2019-2025年年度报告及业绩预告[巨潮资讯网]
```

## 📚 **五、引用格式规范**

### **5.1 文中引用**
- **格式**：[编号] 或 (作者, 年份)
- **位置**：标点符号之前
- **示例**：Yin（2013）[1]提出...

### **5.2 参考文献格式**
**统一标准**：GB/T 7714-2015

| 文献类型 | 格式示例 |
|----------|----------|
| **期刊论文** | [1] 作者. 题名[J]. 期刊名, 年, 卷(期): 起止页码. |
| **专著** | [2] 作者. 书名[M]. 出版地: 出版社, 出版年. |
| **学位论文** | [3] 作者. 题名[D]. 保存地: 保存单位, 年份. |
| **会议论文** | [4] 作者. 题名[C]//会议名称. 出版地: 出版社, 出版年. |
| **电子文献** | [5] 作者. 题名[EB/OL]. (更新日期)[引用日期]. URL |

### **5.3 数据来源引用**
```
数据来源：大悦城控股2024年年度报告[巨潮资讯网]
或
数据来源：华夏大悦城REIT招募说明书[证监会批文+深交所]
```

### **6.1 全文档统一编号**
| 元素 | 编号格式 | 示例 | 位置 |
|------|----------|------|------|
| **表格** | 表X.X | 表4.1 | 表格上方 |

- 表格、图表分别编号
- 公式连续编号

## 📝 **七、正文格式规范**

### **7.1 段落格式**
- **行间距**：1.5倍行距
- **段前段后**：0.5行
- **首行缩进**：2字符
- **对齐方式**：两端对齐

### **7.2 特殊格式**
| 内容 | 格式要求 |
|------|----------|
| **英文** | Times New Roman，小四号 |
| **数字** | 阿拉伯数字，半角 |
| **百分号** | 与数字连写（如：15.67%） |
| **货币单位** | 中文"亿元"，英文"billion CNY" |
| **日期** | 年月日，中文"2026年4月15日" |

### **7.3 标点符号**
| 符号 | 使用要求 |
|------|----------|
| **逗号** | 中文使用全角逗号（，） |
| **句号** | 中文使用全角句号（。） |
| **引号** | 中文使用全角引号（""） |
| **括号** | 中文使用全角括号（（）） |
| **顿号** | 中文使用全角顿号（、） |

## 🎨 **八、视觉设计规范**

### **8.1 颜色使用**
| 用途 | 颜色 | 说明 |
|------|------|------|
| **重点突出** | 黑色加粗 | 不要使用颜色突出 |
| **表格底色** | 浅灰色 | 仅用于表头 |
| **图表线条** | 黑白灰 | 学术论文建议色系 |

### **8.2 空白使用**
| 位置 | 空白要求 |
|------|----------|
| **标题后** | 空一行 |
| **段落间** | 正常行距 |
| **表格上下** | 各空一行 |
| **图表上下** | 各空一行 |

### **8.3 分页要求**
- 表格、图表尽量不跨页
- 新章节从新页开始
- 避免孤行（段落最后一行单独成页）
