# 第三章 研究设计

**本章核心论点**：采用Yin案例研究法作为核心方法论，结合财务指标趋势分析、事件研究法和比较分析法，构建了多方法综合研究范式。研究设计创新性地采用"中长期跟踪验证框架"，将分析期间从REITs发行后的3个月延伸至完整的15个月运营周期，克服了短期数据局限，实现了从"信号验证"到"趋势确认"的方法论突破。

> 版本：v3.0（中长期跟踪验证框架适配）
> 更新时间：2026-04-15
> 核心变更：研究期间延伸至2025年，分析路径从"短期验证"改为"中长期跟踪验证"

## 3.1 方法论框架

### 3.1.1 案例研究法（Yin框架）作为核心方法论

本研究采用Robert K. Yin（2013）[1]的案例研究法作为核心方法论，基于以下设计：

| Yin框架要素 | 本研究应用 | 理论依据 |
|------------|------------|----------|
| **研究问题类型** | "How/Why"型问题：REITs如何缓解财务困境 | 案例研究最适合回答"How/Why"的机制性问题 |
| **理论主张** | 四机制理论框架（融资/现金流/资产负债/运营） | 提供理论预期，指导证据收集 |
| **分析单位** | 大悦城（单案例深度）+ 万科/保利（简要参照） | 大悦城为典型代表案例 |
| **逻辑连接** | 模式匹配（理论预期 vs. 实证发现） | 检验理论框架的解释力 |

**研究问题类型**：本研究核心问题为"REITs如何缓解受困房企的财务困境？"，属于"How/Why"型问题，适合采用案例研究方法进行深入分析。

**理论主张**：基于REITs理论和财务困境理论，构建四机制分析框架（融资结构优化、现金流改善、资产负债重构、运营效率提升），指导证据收集和模式匹配。

**分析单位**：以大悦城控股为单案例进行深度分析，同时以万科、保利为简要参照案例，提供对比视角。

**逻辑连接**：采用模式匹配方法，对比理论预期与实证发现的一致性程度，检验四机制框架的解释力。

### 3.1.2 方法组合设计

研究采用多层次方法组合策略，构建了"核心方法+辅助验证+探索分析"的三层方法体系：

| 方法层级 | 具体方法 | 研究定位 | 数据需求特征 | 主要产出 |
|----------|----------|----------|--------------|----------|
| **核心方法层** | Yin案例研究法 | 理论框架检验与机制深度解析 | 大悦城深度数据，强调纵向时间序列 | 四机制传导路径分析 |
| **辅助验证层** | 财务指标趋势对比 | 财务效应量化验证 | 2019-2025年完整时间序列数据 | 趋势图表与统计结果 |
| **辅助验证层** | 事件研究法 | 市场反应短期效应检验 | 股价高频数据，事件窗口分析 | 累计超额收益率（CAR） |
| **辅助验证层** | 比较分析法 | 横向参照视角提供 | 万科、保利等参照企业数据 | 行业对比与差异化分析 |
| **探索分析层** | DID模型 | 补充性探索分析 | 受限面板数据（样本量约束） | 探索性差异检验结果 |

### 3.1.3 方法论优化说明

**DID定位调整**：原设计中DID作为核心方法，现调整为**探索性补充分析**，需明确标注样本量限制（3家企业×6年=18个观测值不满足大样本要求）。

**案例设计修正**：万科/保利从"控制组"改为"**对比参照案例**"，提供不同困境程度的参照分析，避免规模差异带来的可比性问题。

**分析框架统一**：采用"**四机制为核心+5D为补充**"的统一框架，避免分析维度冲突。

### 3.3.2 数据分析方法

**趋势对比分析**：
- 时间序列分析：纵向对比分析
- 跨期对比：REITs发行前后变化
- 趋势检验：长期趋势、季节性、周期性分析

**描述性统计分析**：
- 均值、标准差、最大值、最小值
- 分布特征分析
- 异常值检测

**探索性DID分析**：
- 样本限制说明：3家企业×6年=18个观测值
- 定位标注：探索性补充分析
- 谨慎解释：考虑统计推断局限性

**模式匹配分析**：
- 理论预期vs实证发现的系统性对比
- 四机制框架各维度的解释力评估
- 一致/不一致的深度解释

**事件研究法（补充详细参数）**：
```
方法定位：验证REITs申报/发行事件的市场反应，补充分析财务效应之外的市场价值效应
市场基准：沪深300指数日收益率（Rmt）
个股收益率：考虑分红再投资的日收益率（Rit）

参数设置：
├── 估计窗口：[-120, -30]交易日（约6个月，用于估计市场模型参数α、β）
├── 事件窗口：[-20, +20]交易日（涵盖申报日、批复日、上市日等关键事件）
├── 事件日：t=0（2024年5月24日，华夏大悦城REIT申报日）
└── 稳健性窗口：[-30, +30]交易日（用于敏感性检验）

异常收益率计算（市场模型法）：
├── ARit = Rit - (α + β × Rmt)
├── 其中α、β由估计窗口数据回归得出
└── CAR = ΣARit（累计超额收益率）

效应评估：
├── 显著性检验：t检验判断CAR是否显著异于零
├── 经济显著性：CAR绝对值大小及占正常收益率比重
└── 稳健性检验：不同窗口期CAR的敏感性分析
```

> **注**：由于大悦城（000031.SZ）为深交所上市公司，采用深证成指或深证综合指数作为市场基准更为精确。本研究采用沪深300指数作为市场基准进行探索性分析，主要结论需谨慎解读。

### 3.3.3 四机制分析框架详细设计（v3.0中长期验证版）

本研究以"四机制"为核心分析框架，以"5D"为补充视角，系统分析REITs对受困房企的变革效应。

**机制一：融资结构优化机制**

```
理论逻辑：REITs发行 → 资产出表/资金回笼 → 降低债务依赖 → 融资成本下降
检验指标：资产负债率变化、融资成本率变化、债务期限结构变化、有息负债率变化
数据支撑：大悦城2019-2025年负债数据 + 华夏大悦城REIT说明书
预期发现：REITs发行后资产负债率下降，融资成本率改善
中长期验证：
  ├── ✅ 2025H1融资成本率下降（财务费用/总负债比率改善）
  ├── ✅ 2025H1有息负债率下降（有息负债/总资产≈25.98%）
  └── 🔄 资产负债率仍处高位（75.87%），需2025全年确认趋势
```

**机制二：现金流改善机制**

```
理论逻辑：REITs发行 → 一次性现金流入 + 持续分红收入 → 经营现金流改善
检验指标：经营现金流比率、现金持有水平、筹资现金流变化、REITs分派贡献
数据支撑：大悦城2019-2025年现金流量表 + 华夏大悦城REIT运营报告
预期发现：REITs发行后经营现金流比率改善，REITs分派贡献持续稳定
中长期验证：
  ├── ✅ 2025H1经营现金流71.17亿元（同比+7.6%，2024H1为66.17亿全年口径需调整）
  ├── ✅ 华夏大悦城REIT 2025Q1年化分派率5.46%，分派金额4,473.76万元
  ├── ✅ 2025H1出租率98%+，租金收入稳定
  └── ⚠️ 筹资现金流仍为净流出，REITs对筹资端改善有限
```

**机制三：资产负债重构机制**

```
理论逻辑：REITs发行 → 重资产出表 → 轻资产转型 → 资产周转率提升
检验指标：资产周转率变化、资本结构变化、偿债能力指标变化
数据支撑：大悦城2019-2025年资产负债表 + REITs资产剥离信息
预期发现：REITs发行后资产周转率提升，资产负债结构改善
中长期验证：
  ├── 🔄 2025H1总资产1,770.27亿元（vs 2024年末1,785.75亿，微降）
  ├── 🔄 资产出表效应需2025全年数据确认
  └── ⚠️ 资产出表是双刃剑：降低负债但也减少合并报表优质资产
```

**机制四：运营效率提升机制**

```
理论逻辑：REITs专业化运营 → 底层资产运营效率提升 → 租金收入增长
检验指标：出租率变化、平均租金变化、管理费用率变化、REITs运营指标
数据支撑：华夏大悦城REIT 2024-2025年运营报告 + 成都大悦城运营数据
预期发现：REITs管理下出租率维持高位，运营效率提升
中长期验证：
  ├── ✅ 华夏大悦城REIT 2025H1出租率98%+（持续高位）
  ├── ✅ 2025Q1收入8,469.91万元，年化分派率5.46%
  ├── ✅ 2025H1收缴率维持高水平
  └── 🔄 专业化运营效应需跨年度对比确认
```

### 3.3.4 稳健性检验策略

**敏感性分析**：
| 检验类型 | 具体方法 | 检验目的 |
|----------|----------|----------|
| **变量替换** | 替换衡量同一概念的指标 | 检验结果对变量的敏感性 |
| **时间窗口调整** | 改变事件窗口长度 | 检验结果的时间稳定性 |
| **样本调整** | 排除特定样本或期间 | 检验结果的样本稳健性 |
| **模型设定检验** | 尝试不同函数形式 | 检验模型设定对结果的影响 |

**稳健性检验方案**：
1. **替代变量检验**：使用不同财务指标衡量相同概念
2. **样本子集检验**：分析不同样本子集的稳定性
3. **模型设定检验**：尝试不同回归函数形式
4. **时间窗口检验**：调整事件窗口长度和估计期
5. **异常值处理**：排除极端值检验结果稳定性

### 3.4.2 证据三角形验证策略

```
              证据三角形
             /            \
    定量数据               定性数据
    （财务指标）         （文本材料）
        |                    |
   年报/季报             公告/研报
   REITs说明书           管理层讨论
   行业统计数据            政策文件
        |                    |
   ┌─────────────────────────┐
   │       交叉验证与整合分析        │
   └─────────────────────────┘
```

**验证原则**：
1. **官方优先原则**：优先采用官方财报数据
2. **多源验证原则**：每个数据点至少两个独立来源验证
3. **证据整合原则**：定量+定性证据交叉验证

**验证流程**：
```
数据收集 → 初步筛查 → 来源确认 → 一致性检查 → 最终确定 → 引用准备
```
