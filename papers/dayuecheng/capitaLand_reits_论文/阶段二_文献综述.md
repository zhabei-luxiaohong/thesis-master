## 三、文献综述初稿框架（主题式组织）🔬

### 3.1 组织原则
- **按主题组织，不按时间线**
- **每组文献回答一个子问题**
- **已知→未知→本研究定位**

### 3.2 综述结构

```
2.1 REITs研究综述
    2.1.1 REITs制度研究（L001-L005）
        已知：新加坡/美国REITs制度成熟；中国REITs快速发展
        未知：外资REITs如何适应中国制度环境？
        本研究定位：填补外资REITs中国制度适应研究空白
    
    2.1.2 REITs运营管理（L016-L022）
        已知：REITs运营效率影响因素明确
        未知：跨国REITs运营本土化机制
        本研究定位：提出四条本土化运营路径
    
    2.1.3 REITs资产循环（L023-L028）
        已知：资产循环对REITs回报的积极作用
        未知：外资REITs在中国市场的资产循环策略
        本研究定位：凯德"四进四出"策略深度分析
    
2.2 制度理论与跨国本土化（L006-L010, L029-L035）
    2.2.1 制度理论框架（L006-L010）
    2.2.2 外资金融机构中国本土化（L029-L035）
        已知：金融开放加速外资进入
        未知：REITs领域的本土化路径
        本研究定位：制度理论在REITs领域的应用
    
2.3 研究空白与本文定位
    - 外资REITs中国运营研究极度缺乏
    - 跨制度REITs比较研究不足
    - 首单外资C-REIT申报案例的时效性
    → 本文贡献：首次系统分析外资REITs中国本土化
```
