## 二、推荐工作流架构

### **核心组件**：
```
1. 模板引擎：LaTeX（SJTUThesis 或 ChineseResearchLaTeX）
2. 版本控制：Git + GitHub
3. 自动化：GitHub Actions
4. 协作平台：GitHub Issues + Projects
5. 写作工具：Visual Studio Code + LaTeX插件
```

### **工作流步骤**：

#### **第1步：项目初始化**
```bash
# 克隆LaTeX模板
git clone https://github.com/huangwb8/ChineseResearchLaTeX.git thesis

# 初始化Git仓库
cd thesis
git init
git add .
git commit -m "Initialize LaTeX thesis template"
```

#### **第2步：GitHub设置**
1. 创建GitHub仓库
2. 配置Git远程仓库
3. 设置GitHub Actions工作流

#### **第3步：自动化编译配置**
```yaml
# .github/workflows/compile.yml
name: LaTeX Compile
on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Compile LaTeX
        uses: xu-cheng/latex-action@v2
        with:
          root_file: main.tex
          latexmk_use_lualatex: true
```

#### **第4步：协作工作流**
1. **Issue跟踪**：每个章节一个Issue
2. **分支管理**：feature/章节名 分支
3. **代码审查**：Pull Request + Review
4. **自动化测试**：编译检查 + 格式检查

### **3. 工作流配置**

#### **3.1 GitHub Actions配置文件**：
```yaml
name: Thesis Build
on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]
  workflow_dispatch:            # 允许手动触发

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v3
        
      - name: Setup LaTeX
        uses: xu-cheng/latex-action@v2
        with:
          root_file: main.tex
          latexmk_use_lualatex: true
          latexmk_shell_escape: true
          extra_system_packages: "texlive-fonts-extra"
          
      - name: Upload PDF
        uses: actions/upload-artifact@v3
        with:
          name: thesis-pdf
          path: build/main.pdf
          
      - name: Save to Release
        if: github.event_name == 'push' && github.ref == 'refs/heads/main'
        uses: softprops/action-gh-release@v1
        with:
          files: build/main.pdf
          tag_name: v1.0.${{ github.run_number }}
```

#### **3.2 LaTeX主文档**：
```latex
\documentclass[12pt,a4paper]{article}
\usepackage[UTF8]{ctex}
\usepackage{geometry}
\usepackage{graphicx}
\usepackage{amsmath}
\usepackage{hyperref}

\title{REITs对受困房企财务困境的变革效应研究}
\author{江峰}
\date{\today}

\begin{document}
\maketitle

\input{chapters/01_introduction}
\input{chapters/02_literature}
\input{chapters/03_methodology}
\input{chapters/04_results}
\input{chapters/05_discussion}
\input{chapters/06_conclusion}

\bibliographystyle{plain}
\bibliography{bibliography}

\end{document}
```

### **4. 自动化任务**

#### **4.1 编译自动化**：
- 每次Push自动编译LaTeX
- 生成PDF并保存为Artifact
- 创建版本标签和Release

#### **4.2 质量检查**：
```yaml
# .github/workflows/quality.yml
name: Quality Check
on: [push, pull_request]

jobs:
  check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Check spelling
        uses: rojopolis/spellcheck-github-actions@v0.35
        with:
          config_path: .github/spellcheck-config.yml
          
      - name: Lint LaTeX
        uses: textlint/textlint-action@v1
        with:
          config_path: .textlintrc.json
```
