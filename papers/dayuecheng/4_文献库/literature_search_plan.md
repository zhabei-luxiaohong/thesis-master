## 3. 文献去重与初步筛选

### 3.1 去重标准（4月17日下午）

#### 去重方法：
1. **自动去重**：使用文献管理软件（Zotero/EndNote）自动去重
2. **人工检查**：检查标题、作者、年份的相似性
3. **合并记录**：同一文献在不同平台的记录合并

#### 去重标准：
- 相同标题、相同作者 → 保留高质量版本
- 相同内容、不同语言 → 保留英文版本（如有）
- 预印本和正式发表版 → 保留正式发表版

### 3.2 初步筛选标准

#### 筛选流程：
```
原始文献 → 标题筛选 → 摘要筛选 → 全文初筛 → 初选文献
```

#### 筛选标准：
1. **标题筛选**（快速排除）：
   - 排除明显不相关的文献
   - 排除非学术性文献（新闻报道等）
   - 排除质量明显偏低的文献

2. **摘要筛选**（重点评估）：
   - **相关性**：是否涉及REITs、财务困境、房企等核心概念
   - **质量**：研究方法是否严谨、数据是否可靠
   - **创新性**：是否有理论或方法创新
   - **时效性**：是否为近5年文献（经典文献除外）

3. **全文初筛**（快速浏览）：
   - 浏览引言和结论部分
   - 检查研究方法和数据部分
   - 评估文献质量和相关性

### 3.3 筛选结果处理

#### 分类标准：
- **高优先级**：直接相关、高质量、必须精读 → 标记为Q1候选
- **中优先级**：理论相关、质量较高、需要阅读 → 标记为Q2候选
- **低优先级**：间接相关、质量一般、选择性阅读 → 标记为Q3/Q4候选
- **排除**：不相关、质量低、重复 → 排除

#### 目标数量：
- 高优先级文献：20-30篇（Q1候选）
- 中优先级文献：30-40篇（Q2候选）
- 低优先级文献：剩余文献（Q3/Q4候选）
- 总初选文献：100篇
