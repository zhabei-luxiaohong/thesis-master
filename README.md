# Thesis Master Skill

> AI-Powered Academic Writing Assistant
> A reusable AI skill for MBA, Master & PhD thesis writing

## What is Thesis Master?

Thesis Master is an AI skill that transforms how academic papers are written. It provides structured, methodology-driven guidance through every phase of thesis writing — from topic validation to final draft.

## Features

- 🧠 **Smart Planning** — AI generates structured 6-chapter thesis outlines in seconds
- ✍️ **Chapter Generation** — Write each chapter with academic rigor, proper citations, and field-specific terminology
- 📚 **Literature Review** — Q1-Q4 quadrant matrix for systematic literature analysis
- 🔬 **Methodology Design** — Yin case study framework, evidence triangulation, 6D financial analysis
- ⚡ **Interactive Assistant** — Conversational AI that guides you through each phase

## Research Frameworks Included

- **6D Analysis**: Profitability, Solvency, Efficiency, Growth, Cash Flow, Market Value
- **Q1-Q4 Matrix**: Systematic literature categorization
- **Yin Case Study**: Evidence triangulation methodology
- **IMRaD Structure**: Introduction, Methods, Results, and Discussion
