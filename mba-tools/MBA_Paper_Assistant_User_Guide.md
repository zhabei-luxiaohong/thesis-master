# MBA 论文写作助手 - 用户指南

## 🎯 系统概述

本系统是一个集成了 **Academic Writing Assistant** 和 **Scientific Agent Skills** 的专业MBA论文写作平台，特别优化了REITs研究方向的支持。

### 核心优势
- ✅ **专业写作工具**：学术语言优化、引用格式化、文献管理
- ✅ **研究技能库**：133个科学技能，支持复杂研究工作流
- ✅ **MBA专属支持**：REITs分析框架、商业案例研究
- ✅ **本地部署**：数据隐私安全，无需担心数据泄露

## 📝 核心功能详解

### A. Academic Writing Assistant 功能

#### 1. 写作改进工具
- **使用方式**: 粘贴文本，选择改进类型（清晰度、结构、语气、词汇、流畅度）
- **适用场景**: 开题报告、文献综述、讨论部分
- **输出**: 逐条建议，包含修改理由和示例

#### 2. 引用与格式化
- **支持格式**: APA 7th, MLA 9th, Chicago 17th, Harvard, IEEE, Oxford, Vancouver
- **数据来源**: DOI、URL、手动输入
- **批量操作**: 一次性生成完整参考文献列表

#### 3. 校对工具
- **检查项**: 语法、拼写、标点、措辞、一致性
- **评分系统**: 0-100分质量评分
- **对比视图**: 原文与修正版本并排显示

### B. Scientific Agent Skills 集成

#### 1. 核心技能调用
以下技能已可用：

| 技能名称 | 调用方式 | 示例用途 |
|---------|---------|---------|
| **scientific-writing** | 通过助手调用 | 科学论文结构化写作 |
| **literature-review** | 通过助手调用 | 文献综述和批判性分析 |
| **citation-management** | 通过助手调用 | 参考文献管理和验证 |
| **paper-lookup** | 通过助手调用 | 查找特定论文信息 |

#### 2. MBA 论文写作专用技能
- **文件位置**: `.workbuddy/skills/MBA_Thesis_Writing.md`
- **核心功能**: 
  - REITs研究框架（5D分析法）
  - 商业案例分析工具
  - 论文结构规划器
  - 进度管理系统

## 🧪 实用工作流程示例

### 工作流 1: 开题报告撰写

**步骤 1: 研究选题确认**
```
使用 bgpt-paper-search 技能搜索：
"中国基础设施REITs发展现状 2025年研究"
```

**步骤 2: 文献综述准备**
```
使用 literature-review 技能整理：
1. 国际REITs发展经验
2. 中国REITs政策演变
3. 相关理论框架（资本结构、风险管理）
```

**步骤 3: 研究设计**
```
使用 MBA_Thesis_Writing 技能：
1. 设计5D分析框架
2. 确定案例研究方法
3. 制定数据收集方案
```

**步骤 4: 报告撰写**
```
使用 Academic Writing Assistant：
1. 导入文献综述结果
2. 逐章节撰写
3. 格式检查和语言优化
```

### 工作流 2: 案例分析研究

**步骤 1: 案例数据收集**
```
使用 paper-lookup 技能查找：
1. 万科租赁住房REITs招股说明书
2. 相关财务报告和分析
```

**步骤 2: 数据分析**
```
使用 MBA_Thesis_Writing 技能：
1. 财务绩效分析（ROE、ROA、分红率）
2. 风险评估（市场风险、运营风险）
3. 投资回报预测
```

**步骤 3: 结果撰写**
```
使用 scientific-writing 技能：
1. 结果展示（表格、图表）
2. 分析讨论
3. 结论和建议
```

### 工作流 3: 论文修改与完善

**步骤 1: 内容审查**
```
使用 Academic Writing Assistant：
1. 整篇论文语法检查
2. 逻辑结构评估
3. 学术语言质量评分
```

**步骤 2: 引用验证**
```
使用 citation-management 技能：
1. 检查所有参考文献格式
2. 验证DOI链接有效性
3. 生成完整的参考文献列表
```

**步骤 3: 格式最终检查**
```
检查项目：
1. 封面页格式
2. 目录自动生成
3. 页眉页脚设置
4. 图表编号格式
```

## 💡 高效使用技巧

### 技巧 1: 模块化写作
- **分解任务**: 每次只处理一个章节或一个分析模块
- **定期保存**: 使用版本控制（Git）管理论文版本
- **批量操作**: 集中进行引用格式化和数据收集

### 技巧 2: 技能组合使用
**REITs财务分析组合**:
1. `bgpt-paper-search` → 查找行业报告
2. `paper-lookup` → 获取具体案例数据
3. `MBA_Thesis_Writing` → 进行财务分析
4. `scientific-writing` → 撰写分析报告

### 技巧 3: 进度管理
1. **制定详细计划**: 使用论文结构规划器
2. **设置里程碑**: 每周完成目标
3. **定期审查**: 导师反馈整合机制

## 🔧 故障排除指南

### 常见问题 1: Academic Writing Assistant 无法访问
**症状**: 访问 http://localhost:3000 时页面无响应
**解决方案**:
```bash
# 1. 检查服务器是否运行
ps aux | grep "next dev"

# 2. 重启服务器
cd academic-writing-assistant
pkill -f "next dev"
PATH="/Users/op/.workbuddy/binaries/node/versions/22.12.0/bin:$PATH" /Users/op/.workbuddy/binaries/node/versions/22.12.0/bin/npm run dev
```

### 常见问题 2: 技能无法识别
**症状**: 调用技能时助手无法识别
**解决方案**:
1. 确认技能文件位置: `.workbuddy/skills/`
2. 检查文件格式: 必须是 `.md` 格式
3. 重新加载技能配置

### 常见问题 3: 数据访问受限
**症状**: 无法访问某些数据库或API
**解决方案**:
1. 检查网络连接
2. 确认是否需要API密钥
3. 使用替代数据源

## 📊 项目管理建议

### 时间规划模板

| 阶段 | 时间分配 | 主要任务 | 交付成果 |
|------|----------|---------|----------|
| **准备阶段** | 1-2周 | 选题、初步调研 | 研究计划 |
| **开题报告** | 2-3周 | 文献综述、研究设计 | 开题报告 |
| **数据收集** | 3-4周 | 案例研究、数据整理 | 数据集 |
| **分析写作** | 4-6周 | 各章节撰写 | 初稿 |
| **修改完善** | 2-3周 | 格式检查、语言润色 | 终稿 |

### 质量控制检查清单
- [ ] 格式符合《MBA论文写作规范》
- [ ] 引用完整且格式正确
- [ ] 数据和案例分析充分
- [ ] 逻辑结构清晰（引言→文献→方法→结果→讨论→结论）
- [ ] 语言表达专业，无语法错误
- [ ] 图表清晰，标注完整
- [ ] 参考文献覆盖核心文献
- [ ] 创新点和贡献明确
