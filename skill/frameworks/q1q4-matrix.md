# Q1-Q4 Literature Matrix

Systematic literature review framework for categorizing and analyzing academic papers.

## Four Quadrants

```
                  HIGH Relevance
                       │
         Q2            │           Q1
    Important but      │     Core Literature
    not directly       │     (must cite)
    applicable         │
───────────────────────┼───────────────────────
                       │
         Q4            │           Q3
    Background/        │     Methodological
    Context            │     References
                       │
                  LOW Relevance
```

## Quadrant Definitions

| Quadrant | Label | Description | Action |
|----------|-------|-------------|--------|
| **Q1** | Core Literature | Directly relevant, must cite | Deep read & cite |
| **Q2** | Theoretical Foundation | Important theories, indirect relevance | Summarize key points |
| **Q3** | Methodological | Similar methods, different topics | Reference methods |
| **Q4** | Background | General context, low relevance | Brief mention |

## Usage

```yaml
研究主题: REITs轻资产运营模式
搜索关键词: REITs, asset-light, financial performance, real estate

Q1_核心文献:
  - "REITs与商业地产轻资产转型" (2023) - 直接相关
  - "Asset-light Strategy in Real Estate" (2022) - 理论框架

Q2_理论基础:
  - "轻资产运营理论综述" (2021) - 提供分析框架
  - "Resource-based View in Real Estate" (2020) - 理论支撑

Q3_方法论:
  - "双案例对比分析法在财务研究中的应用" (2022) - 方法参考

Q4_背景:
  - "中国商业地产发展报告" (2023) - 行业背景
```
